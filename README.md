# COVID-19 Data Analysis (SQL Project)

## 📌 Project Overview

This project analyzes COVID-19 data using SQL. It includes data cleaning, transformation, and analysis to extract meaningful insights.

## 🔍 Key Analysis

* Total cases and deaths analysis
* Death percentage calculation
* Infection rate by country
* Continent-wise death count
* Global statistics
* Vaccination trends using JOIN
* Rolling vaccination using window functions

## 🛠️ Tools Used

* MySQL
* SQL (Joins, Aggregations, Window Functions)

## 📊 Dataset

COVID-19 dataset (deaths & vaccinations)

## 🚀 Outcome

This project demonstrates SQL skills for real-world data analysis and reporting.
